X-Framework

Сайт документации для фреймворка XedoxFramework.

Введение

Скачайте последнюю версию библиотеки, и поместите файл по дирректории /app/libs/

В файл /app/build.gradle, в блок dependencies добавьте файл в зависимости приложения:

dependencies {
  implementation files("libs/XFW.aar")
}

В MainActivtiy, создайте обьект класса XSurfaceView, передав контекст и наследник класса ScreenAdapter, экраны по своей сути являются сценами.

XSurfaceView view = new XSurfaceView(this, new Main());
setContentView(view);

Создайте главный экранс наследуясь от ScreenAdapter. Также переопределяем методы start() и render(Canvas):

public class Main extends ScreenAdapter {
  @Override
  public void start() {
  }
  
  @Override
  public void render(Canvas canvas) {
  }
}
